# Firebase Auth App

A simple app that showcase authentication flow using firebase made in Flutter and Dart with Cubit State Management
<br />  
### Features:
<ul>  
  <li>Sign In</li>
  <li>Sign Up</li>
  <li>Google Sign In</li>
  <li>Forgot Password</li>  
  <li>User Profile</li>
</ul>
